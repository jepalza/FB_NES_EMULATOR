# FB_NES_EMULATOR
Freebasic emulador de Nintendo (NES) clasica (CPU 6502)

Emulador en FreeBasic de la CPU 6502, con un ejemplo de Nintendo Clásica (NES) y una ROM demo PD (Public Domain).
